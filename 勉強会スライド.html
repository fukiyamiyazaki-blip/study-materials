STEP 1

💻 VS Code + Claude でコードを書く

①

VS Code（エディタ）を開く

コードを書く専用のメモ帳のようなアプリ

②

Claude Code を起動する

VS Codeの中でAIと会話しながら作業できる

③

Claudeに日本語でお願いする

「〇〇できるStreamlitアプリを作って」と入力するだけ

④

app.py というファイルができる

Claudeが自動でコードを書いてくれる

STEP 1 補足

🗣️ Claudeへのお願いの仕方

🙋

こんなふうに
日本語で
話しかけるだけで
OKです！

# 例1：振替休日の管理
「スタッフが休日出勤した日付と振替取得日を入力・登録できて
　申請一覧と未取得者を一覧で確認できる
　Streamlitアプリを作ってください」

# 例2：請求書の一覧表入力
「取引先・請求金額・請求日を入力したら
　Excelの請求一覧表に自動で行を追加してくれる
　Streamlitアプリを作ってください」

# 例3：シフト表の自動作成
「スタッフの希望休・勤務可能日・必要人数を入力したら
　希望をできるだけ反映したシフト表を自動生成してくれる
　Streamlitアプリを作ってください」

💡 ポイント

「Streamlitアプリを作って」と伝えるだけで、そのまま公開できる形で作ってくれます。
うまくいかなければ「〇〇がうまくいかない、直して」と追加で頼めばOKです。

STEP 2

🐙 GitHub にコードを保存する

①

GitHubにリポジトリ（保存場所）を作る

github.com にアクセス → 「New repository」をクリック

②

Claudeに「GitHubにプッシュして」と頼む

コマンド操作もAIが全部やってくれる

③

GitHubにコードが保存される

github.com/ユーザー名/リポジトリ名 でコードが管理される

💡 GitHubって何？

コードのGoogleドライブ。保存・共有・バックアップが全部できます。
変更履歴も残るので「前の状態に戻す」こともできます。

STEP 3

🌐 Streamlit Cloud でアプリを公開する

①

アプリの完成コードをGitHubにプッシュ

各自がClaudeと作ったコードをFukiya-AIのリポジトリへ

②

【初回のみ】宮崎さんがStreamlitで「Create app」

リポジトリを選んで最初の1回だけデプロイ設定をする

③

以降はGitHubプッシュだけで自動反映

コードを更新してプッシュするだけ → 宮崎さんへの依頼不要！

④

URLが発行 → 全員で使える

そのURLをチームに共有すれば、誰でもブラウザで使える
