📋 準備の進捗 0 / 7 完了
1
📧 Claude Codeの招待メールを承認する
宮崎から届いたメールで登録します（会社のメールアドレスを使用すること）
💡
ログインすると「Claude」と会話できる画面が表示されれば成功です。
このアカウントを勉強会でも使います。
2
🐙 GitHubアカウントを作成する
コードを保存・管理する場所です
🔗 https://github.com
⚠️
ユーザー名は後から変更しにくいです。ローマ字のフルネームや会社名+名前など、わかりやすいものにしましょう。
例：yamada-fukiya、fukiya-sato
3
💻 VS Codeをインストールする
コードを書く専用のエディタです（無料）
🔗 https://code.visualstudio.com
4
🤖 Claude Code をインストールする
VS Codeに追加するだけで、AIと話しながらアプリが作れます
⚠️
似た名前の拡張機能が複数表示されます。必ず「Anthropic」が作ったものを選んでください。
正しいものは、アイコンが オレンジ色の「<>」マーク で、作者欄に Anthropic と書いてあります。
💡
インストール後、Claudeアイコンをクリックして
STEP1で会社メールアドレスで登録したClaudeアカウント（claude.ai）でログインしてください。
これで Claude Code が使えるようになります！
5
🐍 Pythonをインストールする
アプリを動かすために必要なプログラムです
🔗 https://www.python.org/downloads/
⚠️
インストール画面の一番下にある
「Add Python to PATH」に必ずチェックを入れること！
これを忘れると次のステップができません。チェックを入れてから「Install Now」をクリック。
6
🌐 Streamlitをインストールする
アプリを作るための部品です（コマンドを1行入力するだけ）
pip install streamlit
💡
インストール中は黒い画面にいろんな文字が流れますが、正常です。
止まるまでそのままお待ちください（1〜3分程度）。
7
🏢 富喜屋GitHubグループへの招待を承認する
社内チームのコード共有グループに参加します
💡
GitHubにログインしていない状態でリンクをクリックすると承認できないことがあります。
先に github.com でログインしてからリンクをクリックしてください。

🎉 準備完了！

すべての準備ができました。当日お会いしましょう！
困ったことがあれば miyazaki@fukiya-meals.co.jp までご連絡ください。
