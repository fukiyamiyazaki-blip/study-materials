# 富喜屋 AI利用ルール

## ファイル・データの削除について
- ファイルやデータを削除する前に、**必ず日本語で事前確認**を取ること
- 削除許可を得た後も、実際には削除せず、デスクトップの「AI削除用」フォルダにすべて移動すること
- 「AI削除用」フォルダが存在しない場合は作成してから移動すること

## 返答スタイル
- 返答は**日本語で簡潔**に行うこと

## セキュリティ
- APIキー・パスワード・個人情報を含むファイルは絶対にGitHubにプッシュしないこと
- `.env` ファイルや `secrets.toml` は絶対にコミットしないこと
- 外部サービスへのデータ送信は事前に確認を取ること

## 操作の確認（必須）
- 大量のファイルを一括変更・削除する操作は必ず事前確認を取ること
- `git push --force` などの破壊的な操作は絶対に実行しないこと
- `git reset --hard` は実行しないこと
- 本番環境（Streamlit公開済みアプリ）への変更は事前に確認を取ること

## データ取り扱い
- 顧客情報・スタッフ情報などの個人情報を含むファイルはGitHubにアップロードしないこと
- テストデータには実際の個人情報を使用しないこと
- 献立データ・仕入データ・シフトデータは社外に送信しないこと
