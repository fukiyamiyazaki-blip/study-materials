PHASE 2
Gitの初期設定をする
⏱ 約5分
3
⚙️ あなたの名前とメールをGitに登録する
GitHubに「誰が変更したか」を記録するための設定です（初回のみ）
git config --global user.name "ここに自分の名前" 例）git config --global user.name "Miyazaki Hiroo"
git config --global user.email "ここにメールアドレス" 例）git config --global user.email "yamada@fukiya-meals.co.jp"
git config --global --list
💡
この設定はパソコンに1回だけ行えばOKです。
次回からは設定不要です。

📂 VS Codeの大切なルール：「フォルダ単位で開く」
